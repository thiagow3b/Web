Curso: Programador Web

O Curso de Programação Web tem por objetivo a capacitação de profissionais para desenvolvimento de sistemas web, atuando em programação front-end ou back-end. O curso tem carga horária de 240 horas e sua grade é composta por conteúdos de Front-end (HTML, CSS e JavaScript) e Back-end (PHP e SQL).


O Senac conta com diversos cursos na área de programação, como: Lógica de Prgramação, Programação de Sistemas, Fullstack, Formação em Python
